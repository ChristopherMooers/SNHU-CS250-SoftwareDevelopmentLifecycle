# SNHU-CS250-SoftwareDevelopmentLifecycle

This repository contains coursework, exercises, and portfolio artifacts for  
CS-250: Software Development Lifecycle at Southern New Hampshire University.

## Topics Covered
- Software Development Life Cycle (SDLC)
- Agile and Scrum practices
- User stories and acceptance criteria
- Requirements gathering and analysis
- Stakeholder communication
- Software planning and iteration

## Purpose
This repository is part of my academic portfolio and demonstrates my understanding
of modern software development processes and Agile methodologies.
